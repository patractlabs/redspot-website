## What

This plugin brings to Hardhat the Ethereum library ethers.js, which allows you to interact with the Ethereum blockchain in a simple way.

## Installation

```shell
npm install --save-dev @nomiclabs/hardhat-ethers 'ethers@^5.0.0'
```
